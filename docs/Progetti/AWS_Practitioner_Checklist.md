# AWS Certified Cloud Practitioner (CLF-C02) Checklist

Usa questa checklist insieme al [Piano Decennale](./AWS_Cloud_Practitioner_10Day_Plan.md) per monitorare i tuoi passi verso il 2 Maggio.

---

## ☁️ Dominio 1: Cloud Concepts (24%)
- [ ] Definire il valore della AWS Cloud (Agilità, Costi, Scalabilità).
- [ ] Identificare gli aspetti dell'economia del Cloud (TCO, OpEx vs CapEx).
- [ ] Spiegare i principi di design del Cloud (Design for failure, Decoupling).

## 🔐 Dominio 2: Security and Compliance (30%)
- [ ] Capire lo **Shared Responsibility Model**.
- [ ] Concetti di **IAM** (Utenti, Gruppi, Ruoli, Policy, MFA).
- [ ] Risorse di conformità (AWS Artifact).
- [ ] Servizi di sicurezza: WAF, Shield, GuardDuty, Inspector, KMS.

## 🚀 Dominio 3: Cloud Technology & Services (34%)
- [ ] Metodi di distribuzione: Console, CLI, SDK.
- [ ] **Infrastructure:** Regions, Availability Zones, Edge Locations.
- [ ] **Compute:** EC2, Lambda, Fargate, Lightsail.
- [ ] **Storage:** S3 (Classes, Life Cycle), EBS, EFS.
- [ ] **Database:** RDS, Aurora, DynamoDB, Redshift.
- [ ] **Networking:** VPC, Security Groups, NACL, Route 53, CloudFront.

## 💰 Dominio 4: Billing, Pricing, and Support (12%)
- [ ] Modelli di costo (On-demand, Reserved, Spot, Savings Plans).
- [ ] Strumenti di costo: AWS Budgets, Cost Explorer.
- [ ] **Support Plans:** Basic, Developer, Business, Enterprise On-Ramp, Enterprise.

---

*💡 **Focus SNOC:** Molti di questi servizi saranno il tuo pane quotidiano quando monitorerai le infrastrutture cloud dei clienti. Buona fortuna!*
