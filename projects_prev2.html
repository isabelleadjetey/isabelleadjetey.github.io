🌐 Simulazione Client-Server con Intercettazione HTTPS/HTTP

Tool: Kali Linux, Windows 7, BIND9, Apache, Wireshark

Obiettivi:

  • Configurare DNS per epicode.internal → 192.168.32.100
  • Attivare server HTTPS con certificato autofirmato
  • Catturare traffico HTTPS e HTTP; analizzare visibilità e sicurezza

Risultati: Trafﬁco HTTPS criptato (solo handshake TLS visibile), HTTP in chiaro (richieste e risposte leggibili).
